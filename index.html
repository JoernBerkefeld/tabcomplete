jquery-tab-complete v0.2.0

Type lowercase a and press tab multiple times.

Getting started

Include scripts

Start by including jQuery and the plugin.

				<script src="jquery.js"></script>
				<script src="jquery.tabComplete.js"></script>

Activate the plugin

Now call .tabComplete() on your inputs, like this:

				<script>
					// jQuery ready event
					$(function() {
						$('.target').tabComplete({
							words: [] // Array with words to complete
						});
					});
				</script>
